# Tp Backend CRUD API - Gestion de Peliculas

Este es un proyecto API RESTful desarrollada con Node.js, Express y MongoDB
Que permite realizar operacions CRUD (Crear, leer, Actualiza, eliminar) sobre colecciones de usuarios y camiones.

# Tecnologias utilizadas

- Node.js
- Express
- MongoDB + Mongoose
- Postman
